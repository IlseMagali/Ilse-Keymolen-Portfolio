Proyectos

Textiles Castell

Diseño de sitio web para empresa mexicana de textiles elaborado con el uso de HTML 5, CSS3 y Jquery. Realicé una investigación referente a la fábrica de textiles en México para desarrollar un concepto de ésta y construir su sitio.

¡Alértate!

App móvil como proyecto colaborativo para alertarte sobre las zonas de riesgo, si piensas recorrer alguna parte de la ciudad, según tu ubicación. Para la realizaciòn de este proyecto, utilicé HTML5, CSS3 y Javascript, así como el uso de API de geolocalización de GoogleMaps. Proyecto elaborado en colaboración con superNovas.

¡A comer!

Aplicación web que ubica al usuario y a los restaurantes cercanos para ir a comer. Para su realización, utilicé HTML5, el framework de Materialize, CSS3 y JavaScript, así como el uso de API de geolocalización de GoogleMaps.

Lyft México

Propuesta de diseño y funcionalidad para aplicación móvil basada en Lyft USA, los recursos que utilicé fueron HTML5, CSS3, así como Javascript para la funcionalidad de la navegación. Este proyecto fue realizado en colaboración con superNovas

LabCar - Taxi Hiring

Propuesta de diseño de sitio web para pedir taxi dependiendo de tu ubicación. Fue desarrollado con el uso del framework Bootstrap, utilizando HTML5 y CSS3. Llevé acabo también una versión móvil de este proyecto con media queries en CSS3.
